Dark mode

stuff theme Dark mode (Auto)

Example

We recommend the li and button tags for dark mode 

<!--choose one-->
<li class="darkmode"></li>
<button class="darkmode"></button>

Default theme

You can choose one of the default themes

Forced the themes on document <html data-theme="light"> or on any HTML element <main data-theme="dark">

<!doctype html>
<html lang="en" data-theme="light">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <link rel="stylesheet" href="css/stuff.min.css">
    <title>I'm light</title>
  </head>
  <body>
    <main data-theme="dark">
      <h1>Now i change to dark!</h1>
    </main>
  </body>
</html>
