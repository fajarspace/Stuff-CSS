Install Manually, CDN, or NPM.

Download latest stuff and linked to dist/ 

<link rel="stylesheet" href="dist/css/stuff.min.css">
<script src="dist/js/stuff.min.js" integrity="sha256-cn93xNeta0ghDISkWeXqV9V48JhQsaeskMV+4P/P5LA=" crossorigin="anonymous"><script>

Alternatively, you can use unpkg CDN to link stuff css 

<link rel="stylesheet" href="https://unpkg.com/stuffcss@latest/dist/css/stuff.min.css">
<script src="https://unpkg.com/stuffcss@latest/dist/js/stuff.min.js" integrity="sha256-cn93xNeta0ghDISkWeXqV9V48JhQsaeskMV+4P/P5LA=" crossorigin="anonymous"><script>

Install with NPM

> npm install stuffcss
